# 🎯 RT Point — ระบบบันทึกคะแนนความประพฤตินักเรียน

โปรเจกต์ Demo ระบบบันทึกคะแนนการทำความดีเก็บขวดสำหรับโรงเรียนมัธยม (ม.1–ม.6)
พัฒนาด้วย **HTML + CSS + PHP**

---

## ✨ ฟีเจอร์หลัก

- 🔍 **ค้นหานักเรียน** — กรอกรหัสนักเรียน + เลือกชั้น/ห้องเพื่อเข้าระบบ
- ➕ **เพิ่ม / ตัดคะแนน** — บันทึกการเก็บขวดด้วยการกรอกรหัสใต้ฝา
- 📊 **ตรวจสอบคะแนน** — ดูคะแนนสะสมและประวัติการได้คะแนน
- 🎨 **แยก Stylesheet ตามหน้า** — แต่ละหน้ามี CSS ของตัวเอง ดูแลง่าย
- 🏫 **รองรับ ม.1–ม.6** — ครอบคลุม 6 ระดับชั้น (50+ ห้องเรียน)

### 🤔 ทำไมแยกไฟล์แบบนี้?

แต่ละหน้า (`*.html`) จับคู่กับไฟล์ทำงาน 2 ตัว:
- **`Style{ชื่อ}.css`** → ตกแต่งหน้า
- **`Process{ชื่อ}.php`** → รับข้อมูลจาก form ของหน้านั้น

ตัวอย่าง flow ของหน้า home:
```
home.html  →  ส่ง form  →  ProcessHome.php
   ↑               ↓             ↓
StyleHome.css   ตรวจสอบ      redirect ไปหน้าถัดไป
```

## 🔄 โปรเจกต์นี้ทำงานยังไง?

### ขั้นตอนการใช้งาน

```
1️⃣ home.html
    └─ ผู้ใช้กรอก "รหัสนักเรียน" + เลือก "ชั้น/ห้อง"
       ↓ (POST)
    ProcessHome.php
    └─ ตรวจสอบข้อมูล → ส่งต่อไปหน้าถัดไป
       ↓
2️⃣ type.html
    └─ กรอกเลขใต้ฝา
       ↓ (POST)
    ProcessType.php
       ↓
3️⃣ rtpoint.html
    └─ แสดงผล/บันทึกคะแนน
       ↓
4️⃣ check-point.html
    └─ ตรวจสอบคะแนนสะสมของนักเรียน
       ↓ (POST)
    ProcessCheck.php
```

### หน้าทั้ง 4 ทำหน้าที่อะไร?

| หน้า | หน้าที่ |
|------|--------|
| **home.html** | จุดเริ่มต้น ระบุตัวนักเรียน |
| **type.html** | กรอกเลขใต้ฝา |
| **rtpoint.html** | บันทึกและแสดงคะแนน |
| **check-point.html** | ดูสรุปคะแนนของนักเรียน |

## 📖 วิธีใช้งาน

1. เข้าหน้าแรก (`home.html`)
2. กรอก **รหัสนักเรียน** ของตัวเอง
3. เลือก **ชั้น/ห้อง** (เช่น ม.1/1, ม.5/3)
4. กดปุ่ม **"ต่อไป"**
5. กรอกเลขใต้ฝา
6. ระบบจะแสดง/บันทึกคะแนน
7. สามารถกลับมาตรวจสอบคะแนนสะสมได้ที่หน้า **Check Point**
